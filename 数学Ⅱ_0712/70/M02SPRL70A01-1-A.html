1) sin ⁡ 75 ∘ cos ⁡ 15 ∘
2 + 3 4
= 1 2 { sin ⁡ ( 75 ∘ + 15 ∘ ) + sin ⁡ ( 75 ∘ − 15 ∘ ) } = 1 2 ( sin ⁡ 90 ∘ + sin ⁡ 60 ∘ ) = 1 2 ( 1 + 3 2 ) = 2 + 3 4
2) cos ⁡ 45 ∘ cos ⁡ 15 ∘
1 + 3 4
= 1 2 { cos ⁡ ( 45 ∘ + 15 ∘ ) + cos ⁡ ( 45 ∘ − 15 ∘ ) } = 1 2 ( cos ⁡ 60 ∘ + cos ⁡ 30 ∘ ) = 1 2 ( 1 2 + 3 2 ) = 1 + 3 4
