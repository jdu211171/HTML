x 2 + y 2 − 4 x − 6 y = 0
Aylana tenglamasini x 2 + y 2 + l x + m y + n = 0 shaklida olamiz.

A ( − 1 , 1 ) nuqtadan o'tadi. Bundan, ( − 1 ) 2 + 1 2 − l + m + n = 0 − l + m + n + 2 = 0

B ( 5 , 1 ) nuqtadan o'tadi. Bundan, 5 2 + 1 2 + 5 l + m + n = 0 5 l + m + n + 26 = 0 C ( 4 , 6 ) nuqtadan o'tadi.Bundan, 4 2 + 6 2 + 4 l + 6 m + n = 0 4 l + 6 m + n + 52 = 0 Bularni yechib, l = − 4 , m = − 6 , n = 0
