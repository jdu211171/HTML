1) ∫ ( − 5 x + 8 ) d x

∫ ( − 5 x + 8 ) d x = − 5 ⋅ 1 2 x 2 + 8 ⋅ x + C = − 5 2 x 2 + 8 x + C ( C integratsiya doimiysi)

2 x 3 − 11 2 x 2 + 3 x + C
∫ ( 6 x 2 − 11 x + 3 ) d x = 6 ⋅ 1 3 x 3 − 11 ⋅ 1 2 x 2 + 3 ⋅ x + C = 2 x 3 − 11 2 x 2 + 3 x + C
